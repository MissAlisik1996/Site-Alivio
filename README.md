# Приветствую всех!
##### Это мой второй учебный сайт, который  создавая, я оттачивала свои навыки.
###### Какие темы я изучила глубже или впервые:
* **позиционирование элементов**;  
* **резиновые отступы и шрифты, которые очень укорачивают медиа запросы**;  
* **формирование логической структуры HTML файла**;  
* **учет margin-ов по умолчанию у некоторых элементов**;  
* **создание гибкой ширины**.  

###### Не учла:
* **использование переменных**;
* **оптимизирование изображений с помощью современных форматов**;
* **не добавлен файл normalize**.
